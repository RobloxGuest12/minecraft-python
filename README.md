# Pygame Dual Mode Project

This project is a simple Pygame-based game setup with **two launch modes**:

---

## Normal Mode (`main.pyw`)
- Runs the game normally
- No console window
- Clean player experience
- Used for playing the game

---

## Debug Mode (`debug.py`)
- Shows console logs and debug info
- Helps track events and errors
- Displays on-screen debug text
- Used for development and testing

---

## How to Use

### Run Normal Game:
Double-click:
```
main.pyw
```

### Run Debug Version:
Run:
```
debug.py
```

---

## Purpose
This setup helps separate:
- Player experience (clean game)
- Developer experience (debug tools)

---

## Notes
- Built using Python + Pygame
- Designed for easy testing and development
- You can expand both modes without rewriting the whole game
